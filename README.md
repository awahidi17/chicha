# 🚀 Space Explorer

Welcome to **Space Explorer** — a project dedicated to exploring and sharing information about the universe, space missions, celestial bodies, and more.

## 🌌 About This Project

Space Explorer is a web-based project that provides users with fascinating data and visuals related to space. Whether you're a student, developer, or just curious about space, this project brings cosmic knowledge closer to Earth.

## 🔧 Features

- 🌍 Real-time data from space APIs (e.g. NASA, Open Notify)
- 🪐 Information on planets, stars, and galaxies
- 🚀 Space mission history & upcoming launches
- 📸 Space image gallery and astronomy picture of the day
- 🔭 Educational resources about space science

## 🛠️ Built With

- HTML, CSS, JavaScript
- [NASA API](https://api.nasa.gov)
- VS Code
- Git & GitHub
